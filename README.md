# Employee Management API

This is a simple ASP.NET Core 8 Web API project built using Entity Framework Core and SQL Server.

## Features
- Create Employee
- Read Employees
- Update Employee
- Delete Employee
- Swagger API Documentation

## Tech Stack
- ASP.NET Core 8
- Entity Framework Core
- SQL Server (LocalDB)
- Swagger

## API Endpoints

- GET /api/employees
- GET /api/employees/{id}
- POST /api/employees
- PUT /api/employees/{id}
- DELETE /api/employees/{id}

## How to Run
1. Clone repository
2. Update connection string in appsettings.json
3. Run migrations
4. Run project using Visual Studio
